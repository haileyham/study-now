## <span id="1">📖 서비스 소개</span>

**studyNow** 는 공부 등 자기 계발을 위해 설계된 다기능 플랫폼입니다. 자기계발을 위한 퀴즈, 효과적인 협업을 위한 스터디그룹 모집 및 채용정보 제공, 구성원 간 원활한 소통을 위한 실시간 채팅 기능을 갖춘 종합 플랫폼입니다.

1.  **퀴즈 플레이를 통한 개인 실력 향상**

    - 개인의 기술과 지식을 향상시키기 위해 다양한 퀴즈 주제에 도전하고 시간 경과에 따른 진행 상황을 확인할 수 있습니다.
      </br>

2.  **스터디 그룹을 통한 공통 학습 목표 달성 및 협업 능력 향상**

    - 스터디 그룹을 게시하고 가입하여 다양한 학습 주제를 탐색하고 스터디원와 협력하여 공통 학습 목표를 달성할 수 있습니다.
      </br>

3.  **Chatting 서비스를 통해 스터디원과 자유롭게 대화**

    - 스터디원과의 원활한 소통을 위한 실시간 채팅이 가능합니다.
      </br>

4.  **최신 채용 공고에 대한 최신 정보를 유지하고 구직 과정을 간소화** - 서울시 채용정보 연계를 통한 채용 정보에 대한 최신 정보 및 구직 과정을 간소화 시킬 수 있습니다.

## 🎃 props array

### ▶ 요약

퀴즈 플레이 페이지에 props가 array에 제대로 들어오지 않는 문제 발생

### ▶ 어떤 문제?

- props로 받아온 숫자만큼 array 생성이 안 되는 문제
- quizUserSetting 쪽에서 quizPlay로 보내는 props 값 중에 questionNum으로 받는 number 타입의 문제 갯수에서 문제
- quiz user setting에서 quiz play page로 보내는 props type 조정에도 array에 제대로 들어오지 않는 문제

### ▶ 해결 방안

- (놓친 부분)
  1. Array에 props 값이 들어올 때는 단일 요소로 들어오게 됨
  2. questionNum props를 number type으로 받았기에 당연히 숫자로 들어올 줄 알았던 것에 대한 실수
  3. Array(questionNum)이 결국 1개의 배열요소만 배출
  4. 일단 props 자체는 number type이 맞지만, Array에 받을 때는 단일요소로 들어가기 때문에 [’5’]로 받게 됨
- props Number 감싸서 Array 생성하니 제대로 동작 완료

## 🎃 sticky

### ▶ 요약

채용정보 페이지 검색창 스크롤바 내릴 시, sticky 적용 불가 문제

### ▶ 어떤 문제?

- 검색창에 적용한 sticky가 제대로 동작하지 않는 문제
- sticky 적용한 부모에 height, 혹은 body 밑에 바로 걸고 사용해도 제대로 동작 안하는 문제

### ▶ 해결 방안

- 해답은 부모요소들 중에 overflow: hidden, auto, scroll 속성이 적용되어 있으면 안된다는 것
- 부모 요소중에 적용한 것이 없는데 계속해서 문제가 발생하는 것이 이상하여 혹시하고 global.scss를 확인해보니 기본초기 셋팅값이라고 overflow-x:hidden; 을 넣어뒀던 것을 발견하여 수정 완료 후 정상 작동 확인
